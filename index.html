Packing/Unpacking
❮ ❯

This project was made with django and Postgresql to showcase using authentication and CRUD methods. You can create a profile and add upcoming trips and the bags you will be taking. Then you can pack items into your bags from a database of items that are common while traveling. The 'Size' for bags and items that are packable are completely arbitrary, they do not correlate to any real world measurements, i.e. liters which is common for backpack volume measurement. To take this project further I would allow users to create Items for themselves that they are traveling with, alongside commonly brought items. GitHub
